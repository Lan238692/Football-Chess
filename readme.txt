ゲームルール

あなたの目標は、ボールを蹴って相手のゴールに入れることです。
ターン中、自分の選手（赤側）を選んで、パス、シュート、移動のいずれかを選択することができます。
移動は1マスに限られ、ドリブルでディフェンスを交わすときに2行動ポイント、それ以外のときに1行動ポイント消費します。
パスは選手の距離に応じて行動ポイントを消費し、最小で1、最大で3です。パスの距離には上限があります。
シュートは3行動ポイント消費します。シュートの成功率は、シュートする選手の距離と、シュートを阻む敵選手の人数によって変わります。
ターンごとに行動ポイントの数が増えていき、いずれかのチームがゴールを決めるまで続きます。
